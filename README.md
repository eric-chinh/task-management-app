# Task Management App

Full-stack Kanban board — Dự Án 1 trong lộ trình học 6 tháng.

## Stack

**Frontend:** React 18 · Vite · TypeScript · Redux Toolkit · RTK Query · React Router v6 · dnd-kit · Socket.io-client

**Backend:** NodeJS · Express · Prisma · SQLite · JWT · Socket.io

**Deploy:** AWS S3 (frontend) · AWS EC2 + Nginx + PM2 (backend)

## Chạy local

```bash
# Frontend
cd frontend && npm install && npm run dev

# Backend (Tuần 3)
cd backend && npm install && npm run dev
```
